## cleanup

删除目录: node_modules, dist, 等

```bash
sa cleanup
```

## update-pkg

更新 package.json 依赖版本

```bash
sa update-pkg
```

## git-commit

生成符合 Conventional Commits 标准的提交信息

```bash
sa git-commit
```

## git-commit-verify

验证 git 提交信息，确保符合 Conventional Commits 标准

```bash
sa git-commit-verify
```

## changelog

生成 changelog

```bash
sa changelog
```

## release

发布，更新版本，生成 changelog，提交代码

```bash
sa release
```

## gen-route

生成路由

```bash
sa gen-route
```
